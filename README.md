10 coding functions written in aseembly language!

`1` Find the sum of even elements in a matrix.    [ int sum(int *array, int m, int n) ]

`2` Find the max/min of elements in a matrix.    [ int max(int *array, int m, int n) ]

`3` Find the min of elements in a matrix.     [ int min(int *array, int m, int n) ]

`4` Partition an array (as in quick sort). Use first element of array as a pivot. [ void partition(int *array, int noelements) ]

`7` Check if a given 64-bit number contains odd number of 1s in its bit representation [ int parityCheck(int N) ]

`8` Reverse a string [ void reverse(char *str) ]

`9` Merge two sorted lists. [ void merge(int *lista, int sizea, int *listb, int sizeb) ]

`10` Perform binary search for the position of an element in a sorted array.  [ int binarySearch(int *list, int size, int element) ]
